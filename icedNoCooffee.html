Iced No Coffee

Guías de instrucciones estandarizadas para bebidas frías sin café. Asegura medidas consistentes y presentación para cada cliente.

ICED MATCHA

Más Vendido

list_alt Ingredientes Base

  • Leche Fría (M/L/XL) 250 / 330 / 400ml
  • Polvo Matcha 20 / 25 / 30g

info Instrucciones Especiales

"Mezclar por 5s (Botón 1) y verter sobre hielo fresco para asegurar una textura suave y sin espuma."

Iced Chai Latte

Especiado

list_alt Ingredientes Base

  • Concentrado de Chai 3 / 4 / 5oz
  • Leche Fría 160 / 270 / 380ml

info Instrucciones Especiales

"Llenar con hielo (golpear el vaso para asentar), mezclar en jarra primero, luego verter sobre el hielo. Decorar con canela."

SPRITZ REFRESHER

Hielo: 2/3 Taza
Concentrado: Hasta la 2da Línea
Líquido: Agua Gasificada
"Tapar con tapa plana y agitar de lado a lado para mezclar el gas suavemente."

Refresher Limonada

Hielo: 2/3 Taza
Concentrado: Hasta la 2da Línea
Líquido: Base de Limonada
"Tapar con tapa plana y agitar de lado a lado vigorosamente para airear."

Iced Tea

Hielo: 2/3 Taza
Sabores: 3 / 4 / 5 Bombas
Líquido: Base de Té Helado
"Tapar con tapa plana y agitar para combinar sabores y enfriar al instante."
lightbulb

Consejo: Asegúrate siempre de que los vasos estén llenos de hielo exactamente como se especifica para mantener el perfil de sabor adecuado cuando se añaden los líquidos. Golpear el vaso ayuda a eliminar las bolsas de aire.
